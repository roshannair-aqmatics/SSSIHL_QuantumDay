## Experiment Brief: Grover's Algorithm
[cite_start]This experiment demonstrates the **Quantum Advantage** by comparing classical brute-force search against **Grover's Quantum Search Algorithm**[cite: 68, 70]. 

In a classical world, finding an "exit" in an unstructured database takes $O(N)$ attempts. [cite_start]In the Quantum world, we achieve a **quadratic speedup** $O(\sqrt{N})$, finding the needle in the haystack in a single, coherent operation[cite: 39].
